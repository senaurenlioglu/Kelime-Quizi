Kelime-Quizi language quiz application designed to accelerate language learning through active recall and spaced repetition concepts. 

LinguistQuiz was developed to bridge the gap between language acquisition and technology. Inspired by my personal academic journey living and studying abroad, this application helps users master vocabulary in English (B1) and German (B2) through an interactive and timed experience. 

🚀 Key Features

Quiz System: Interactive vocabulary exercises with example sentences to reinforce learning. 


Admin Panel: A secure management interface for adding, editing, and deleting words in the database. 


Statistics Dashboard: Real-time tracking of correct and incorrect answers to monitor learning progress. 


Security & Auth: Robust system featuring JWT-based authentication and role-based (Admin/User) access control. 


Responsive Web Interface: Modern UI designed with React for a seamless experience across devices. 

🛠️ Tech Stack

Frontend: React.js with Context API for global state management and Axios for API requests. 


Backend: Node.js and Express.js providing a RESTful API. 


Database: MongoDB Atlas for secure, cloud-based data storage. 


Authentication: JWT (JSON Web Tokens) and Bcrypt for password encryption. 


Tools: Git, Visual Studio Code, Postman.
