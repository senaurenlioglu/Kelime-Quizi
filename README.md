### 2. Kelime Quiz Projesi  : MERN Stack Vocabulary Builder
Bu dosya staj defterindeki **JWT**, **Admin Paneli** ve **MongoDB** detaylarını içerir.


**An engaging, web-based multi-language quiz application designed to accelerate language learning through active recall and spaced repetition concepts.**

LinguistQuiz was developed to bridge the gap between language acquisition and technology. Inspired by my personal academic journey living and studying abroad, this application helps users master vocabulary in English and German through an interactive experience.

## 🚀 Key Features
* **Quiz System:** Interactive vocabulary exercises with example sentences for context.
* **Admin Panel:** Secure interface for managing the word database (Add/Edit/Delete).
* **Statistics Dashboard:** Tracks correct/incorrect answers and learning progress for both German (B2) and English (B1).
* **Security:** Implemented JWT-based authentication and role-based (Admin/User) access control.

## 🛠️ Tech Stack
* **Frontend:** React.js, Context API (State Management), Axios.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB Atlas (Mongoose ODM).
* **Security:** JWT (JSON Web Tokens), Bcrypt.
